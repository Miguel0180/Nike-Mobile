# Nike-Mobile

Projeto desenvolvido no **FlutterFlow** para demonstrar a criação de um aplicativo mobile inspirado na experiência de navegação da Nike.
O app inclui autenticação, gerenciamento de perfil, cadastro de endereços e uma Home moderna com catálogo de produtos fictícios.


## Telas
<ul>
  <li>Login</li>
  <li>Cadastro</li>
  <li>Endereço</li>
  <li>Home</li>
  <li>Dados do Usuário</li>
  
</ul>
